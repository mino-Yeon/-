BOM이란?

BOM(Browser Object Model)은 브라우저 환경에서 JavaScript의 최상위 전역 객체인 window를 통해 브라우저 자체를 제어할 수 있는 모델입니다.

window는 브라우저 탭 하나를 대표하는 객체로, 모든 전역 변수와 함수는 사실 window의 프로퍼티입니다. 따라서 window.alert()는 alert()과 동일합니다.

window.alert("안녕하세요");
alert("안녕하세요"); // 위와 동일 — window. 생략 가능
window 아래에는 메소드(alert, setTimeout 등)뿐 아니라 location, history, navigator 같은 하위 객체도 있습니다.

alert()

alert

사용자에게 메시지를 보여주는 알림 창을 띄웁니다. 확인을 누를 때까지 이후 코드는 실행되지 않습니다.

alert("저장되었습니다.");
🔔 데모 — alert

confirm()

confirm

확인 / 취소 버튼이 있는 창을 띄우고, 사용자의 선택에 따라 true 또는 false를 반환합니다.

const result = confirm("정말 삭제하시겠습니까?");
console.log(result); // 확인 → true, 취소 → false
✅ 데모 — confirm
버튼을 눌러 결과를 확인하세요.

prompt()

prompt

텍스트 입력창을 띄우고 사용자가 입력한 값을 문자열로 반환합니다. 취소하면 null을 반환합니다.

const name = prompt("이름을 입력하세요.");
console.log(name); // 입력한 문자열 or null(취소 시)
✏️ 데모 — prompt
버튼을 눌러 결과를 확인하세요.

open()

open

새 브라우저 창(또는 탭)을 엽니다. 열린 창에서는 window.opener를 통해 부모 창의 window 객체에 접근할 수 있습니다.

// 새 창 열기
window.open("https://example.com", "_blank");
// 열린 창(자식)에서 부모 창 접근
window.opener.document.title = "부모창 제목 변경";
window.open()은 팝업 차단 정책에 의해 사용자 동작(클릭 등) 없이 호출하면 브라우저가 차단할 수 있습니다.
🪟 데모 — open

setTimeout()

setTimeout clearTimeout

일정 시간(ms) 후에 콜백 함수를 한 번 실행합니다. clearTimeout()으로 실행 전에 취소할 수 있습니다.

const timer = setTimeout(() => {
  console.log("3초 후 실행");
}, 3000);

// 취소
clearTimeout(timer);
⏱ 데모 — setTimeout (3초 후 메시지 표시)
시작 버튼을 누르면 3초 후 메시지가 표시됩니다.

setInterval()

setInterval clearInterval

일정 시간(ms) 간격으로 콜백 함수를 반복 실행합니다. clearInterval()로 중단할 수 있습니다.

const timer = setInterval(() => {
  console.log("1초마다 실행");
}, 1000);

// 중단
clearInterval(timer);
🔄 데모 — setInterval (1초마다 카운트)
00 : 00

location

location

현재 창의 URL 정보를 담고 있는 객체입니다. URL 조회, 페이지 이동, 새로고침 등을 처리할 수 있습니다.

현재 URL 조회

console.log(location.href); // 현재 페이지의 전체 URL

페이지 이동

location.href = "https://example.com"; // history에 기록 남음
location.replace("https://example.com"); // history에 기록 안 남음

새로고침

location.reload();
href로 이동하면 뒤로 가기가 가능하지만, replace()로 이동하면 이전 페이지가 history에 남지 않아 뒤로 가기가 불가합니다. 로그인 후 리다이렉트 등에 활용합니다.
🔗 데모 — 현재 URL 조회
—

history

history

브라우저의 방문 기록을 다루는 객체입니다. 실제 페이지 이동 없이 앞/뒤 이동을 제어할 수 있습니다.

뒤로 가기 / 앞으로 가기

history.back();    // 뒤로 가기 (history.go(-1)과 동일)
history.forward(); // 앞으로 가기 (history.go(1)과 동일)

go(n) — 숫자로 이동

history.go(-2); // 2단계 뒤로
history.go(1);  // 1단계 앞으로
back() / forward()는 go(-1) / go(1)의 축약입니다. 여러 단계를 한 번에 이동할 때는 go(n)을 사용합니다.

navigator

navigator

브라우저와 운영체제에 대한 정보를 담고 있는 객체입니다. 사용자 환경을 파악할 때 활용합니다.

navigator.userAgent
브라우저 및 OS 정보를 담은 문자열
console.log(navigator.userAgent);
—
navigator.language
사용자의 브라우저 언어 설정
console.log(navigator.language);
—
