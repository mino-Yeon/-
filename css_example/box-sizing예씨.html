예시 1 — 아이콘 달린 버튼

.btn::before { width: 28px; padding: 0 4px; border-right: 2px solid; }

::before의 width: 28px 안에 padding + border가 포함되어 버튼 레이아웃이 정확히 맞아떨어집니다.

예시 2 — 체크리스트 아이콘

.checklist li::before { width: 20px; height: 20px; border: 2px solid; padding: 2px; }

  • box-sizing 개념 이해하기
  • content-box vs border-box 차이 파악
  • 전역 border-box 설정 적용해보기
  • ::before / ::after도 박스를 가진다는 것 기억하기

체크박스 크기를 20×20px로 딱 맞게 유지합니다. border + padding이 내부에서 처리됩니다.

예시 3 — border-box vs content-box 직접 비교

::before { width: 6px; padding: 0 1px; } 를 적용했을 때

✅ border-box

전체 width: 120px 유지됨

CSS 학습
❌ content-box

::before의 padding이 밖으로 튀어나옴

CSS 학습
