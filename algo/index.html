반복문(for)을 사용하여 중복순열을 생성하고, 배열(int[])에 결과를 저장하는 방식
iterative

🔄 순열 (Permutation)

반복문 + 배열 (인덱스 비교)
반복문(for)과 인덱스 비교(i==j)로 중복을 제거하여 순열을 생성하는 방식
iterative
반복문 + 배열 + boolean
반복문(for)과 boolean 배열(isSelected)로 중복을 제거하여 순열을 생성하는 방식
iterative
재귀 + boolean
재귀 호출과 boolean 배열(isSelected)를 사용하여 순열을 생성하는 최종 형태
recursive
Next Permutation (단계별)
np() 함수의 4단계를 3가지 예시로 집중 학습 - 포인터와 색상으로 각 단계를 직관적으로 시각화
step-by-step

🔢 조합 (Combination)

반복문 + 배열
반복문(for)과 j=i+1, k=j+1 패턴으로 조합을 생성하는 방식
iterative
재귀 + start 매개변수
재귀 호출과 start 매개변수로 범위를 제한하여 조합을 생성하는 최종 형태
recursive

📦 부분집합 (Subset)

재귀 + boolean
재귀 호출과 isSelected 배열로 각 요소의 포함/비포함을 결정하는 이진 트리 구조
recursive
