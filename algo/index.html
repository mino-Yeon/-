🔀 중복순열 (Duplicate Permutation)

반복문 + List
반복문(for)을 사용하여 중복순열을 생성하고, List에 결과를 저장하는 방식
iterative
반복문 + 배열
반복문(for)을 사용하여 중복순열을 생성하고, 배열(int[])에 결과를 저장하는 방식
iterative
재귀 + List
재귀 호출을 사용하여 중복순열을 생성하고, List에 결과를 저장하는 방식
recursive
재귀 + 배열
재귀 호출을 사용하여 중복순열을 생성하고, 배열(Array)에 결과를 저장하는 방식
recursive
