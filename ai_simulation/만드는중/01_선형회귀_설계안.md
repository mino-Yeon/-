## 3. UI 컴포넌트 목록

1. **상단 헤더**: 페이지 제목 + 한 줄 정의 + `← 목록으로` 뒤로가기 링크
2. **캔버스 영역**: 점 찍기 + 회귀선 + 오차선 표시
3. **MSE 수치 표시판**: 현재 MSE 값 실시간 표시
   - MSE 계산식 표시: `MSE = Σ(실제값 - 예측값)² / n` (고정, 회색 텍스트)
   - 계산식 아래 한 줄 설명: "각 점과 선 사이 거리를 제곱해서 평균낸 값입니다"
   - 계산식 아래 현재 MSE 수치 표시 (강조색 `#6366F1`)
4. **슬라이더 패널**: 기울기(slope) 슬라이더 + 절편(intercept) 슬라이더
5. **최적선 보기 버튼**: 정규방정식으로 계산된 최적선으로 즉시 이동
6. **초기화 버튼**: 캔버스의 모든 점 제거, 슬라이더 초기값 복귀
7. **하단 설명 문구**: 개념 보충 텍스트

## 5. 인터랙션 흐름

### 기본 흐름
1. 사용자가 캔버스 위를 클릭 → 파란 점 추가
2. 점이 추가될 때마다 정규방정식으로 최적선 즉시 계산 → 빨간 회귀선 업데이트
3. 각 점에서 회귀선까지 수직 오차선(연한 회색) 동시 업데이트
4. MSE 수치판 실시간 업데이트
5. 사용자가 슬라이더를 조작 → 회귀선이 슬라이더 값에 따라 이동
6. 슬라이더 조작 시 MSE는 "현재 슬라이더 선의 오차"로 업데이트
7. [최적선 보기] 클릭 → 슬라이더가 최적값으로 이동하며 선이 최적 위치로 이동
8. [초기화] 클릭 → 점 전부 제거, 슬라이더 초기값 복귀, MSE 초기화

### 슬라이더 범위
- 기울기: -5.0 ~ 5.0, 스텝 0.1
- 절편: 캔버스 Y축 범위 기준으로 설정

## 6. 엣지 케이스

| 상황 | 처리 방식 |
|------|-----------|
| 점이 0개 | 회귀선/오차선 미표시, MSE "—" 표시 |
| 점이 1개 | 회귀선 미표시, MSE "—" 표시, 안내 문구: "점을 2개 이상 찍어보세요" |
| 모든 점이 수직 일직선 | 회귀선 수평 표시 또는 "계산 불가" 안내 |
| 캔버스 경계 밖 클릭 | 점 추가 무시 |
| 점 20개 초과 | 추가 무시, "최대 20개까지 찍을 수 있어요" 안내 |

## 7. 표시할 데이터 및 상태값

| 상태값 | 설명 |
|--------|------|
| `points` | 캔버스 위 점들의 좌표 배열 |
| `slope` | 현재 회귀선의 기울기 |
| `intercept` | 현재 회귀선의 절편 |
| `mse` | 현재 선 기준 MSE 값 (소수점 2자리) |
| `optimalSlope` | 정규방정식으로 계산된 최적 기울기 |
| `optimalIntercept` | 정규방정식으로 계산된 최적 절편 |

## 8. 설명 문구

| 위치 | 문구 |
|------|------|
| 캔버스 상단 (작은 회색 텍스트) | "캔버스를 클릭해서 점을 찍어보세요" |
| MSE 수치판 계산식 (고정) | `MSE = Σ(실제값 - 예측값)² / n` |
| MSE 수치판 설명 (고정) | "각 점과 선 사이 거리를 제곱해서 평균낸 값입니다" |
| 슬라이더 패널 상단 | "슬라이더로 선을 직접 움직여보세요. MSE가 어떻게 변하나요?" |
| [최적선 보기] 클릭 시 표시 | "정규방정식: 오차가 가장 작은 기울기와 절편을 수식으로 한 번에 계산하는 방법" |
| 하단 설명 문구 | "선형회귀를 푸는 방법은 두 가지입니다 — 정규방정식(한 번에 계산)과 경사하강법(조금씩 찾아가기). 다음 페이지에서 경사하강법을 직접 체험해보세요." |
