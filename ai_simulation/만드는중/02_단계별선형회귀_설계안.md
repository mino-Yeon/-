## 3. UI 컴포넌트 목록

1. **상단 헤더**: 페이지 제목 + 한 줄 정의 + `← 목록으로` 뒤로가기 링크
2. **캔버스 영역**: 고정된 데이터 포인트 + 현재 회귀선 표시
3. **파라미터 업데이트 카드**: w 카드 + b 카드 나란히 배치
4. **MSE 변화 표시판**: 이전 MSE → 현재 MSE
5. **오차 그래프**: 스텝이 쌓일수록 MSE가 내려가는 꺾은선 그래프
6. **Learning Rate 선택**: select box로 `0.0001 / 0.001 / 0.01` 세 가지 제공, 기본값 `0.001`
   - select box 옆 안내 문구: "값을 바꾸면 학습 속도가 어떻게 달라지는지 확인해보세요"
   - learning rate 변경 시 진행 중인 자동 재생 즉시 중단, [초기화] 자동 실행
7. **컨트롤 버튼**: [다음 스텝] / [자동 재생] / [초기화]
8. **스텝 카운터**: 현재 몇 번째 스텝인지 표시
9. **하단 설명 문구**

## 5. 파라미터 업데이트 카드 상세

### w 카드
- 카드 내부 최상단에 힌트 문구 배치: "매 스텝마다 오차를 줄이는 방향으로 w와 b가 조금씩 바뀝니다"
- 힌트 문구 아래: 수식 (고정, 회색 텍스트 `#6B7280`)
- 수식 아래: 실제 숫자 (매 스텝 업데이트)
- 바뀐 결과값만 강조색 `#6366F1`으로 표시
- 수치: 소수점 3자리 고정

```
w = w - lr × ∂L/∂w
0.817 = 0.82 - 0.01 × 0.34
```

### b 카드
w 카드와 동일한 구조, b 기준으로 표시

## 6. 데이터 설정

- 데이터 포인트: 미리 고정된 10개 점 (사용자 조작 불가)
- 초기 w: `-2.0 ~ 2.0` 사이 랜덤 (페이지 진입 및 [초기화] 시마다 새로 설정)
- 초기 b: `-2.0 ~ 2.0` 사이 랜덤 (페이지 진입 및 [초기화] 시마다 새로 설정)
- Learning rate: `0.01` (고정, 조작 불가)
- 최대 스텝: `100`

※ 랜덤 초기화 범위를 제한하는 이유: 범위가 너무 크면 초기 MSE가 극단적으로 커져 오차 그래프 스케일이 깨질 수 있음

## 6-1. 캔버스 좌표계 정의

### 논리 좌표계 (데이터 공간)
- x축: 좌 → 우 증가
- y축: 아래 → 위 증가 (수학적 표준)
- x 범위: `0.0 ~ 10.0`
- y 범위: `0.0 ~ 10.0`

### 픽셀 좌표계 (캔버스 DOM)
- 브라우저 캔버스는 y축이 위 → 아래 증가 (반전)
- **구현 시 반드시 y축 변환 적용**: `pixelY = canvasHeight - (logicalY / yRange × canvasHeight)`

### 회귀선 렌더링 규칙
- 회귀선은 논리 좌표계 기준으로 계산 후 픽셀 좌표로 변환하여 렌더링
- w(기울기) 양수 → 화면에서 우상향 선으로 표시되어야 함
- w(기울기) 음수 → 화면에서 우하향 선으로 표시되어야 함
- 이 방향이 반전되면 좌표 변환 누락을 의심할 것

### 고정 데이터 포인트 (논리 좌표)

## 7. 인터랙션 흐름

1. **페이지 진입 시 (step=0) 초기 상태**
   - w, b를 `-2.0 ~ 2.0` 사이 랜덤값으로 초기화
   - 캔버스에 랜덤 초기값 기준 회귀선 표시
   - w 카드: 초기 랜덤값 표시, gradW 계산값 표시
   - b 카드: 초기 랜덤값 표시, gradB 계산값 표시
   - MSE 초기값 표시 (prevMse는 표시하지 않음 — 아직 이전 스텝 없음)
   - 오차 그래프: step=0 점 하나만 표시
   - [다음 스텝] 버튼 활성화 상태
2. **[다음 스텝] 클릭** → 1 스텝 진행
   - 캔버스의 회귀선 업데이트
   - w, b 카드 업데이트 (바뀐 결과값 강조색 표시)
   - MSE 이전값 → 현재값 업데이트
   - 오차 그래프에 점 하나 추가
   - 스텝 카운터 +1
3. **[자동 재생] 클릭** → 0.8초 간격으로 자동 스텝 진행
   - 버튼이 [일시정지]로 변경
   - [일시정지] 클릭 시 멈춤
4. **[초기화] 클릭** → 모든 상태 step=0 초기값으로 복귀
5. **100스텝 도달 시** 자동 정지, "학습 완료" 메시지 표시

## 8. 엣지 케이스

| 상황 | 처리 방식 |
|------|-----------|
| 100스텝 도달 후 [다음 스텝] 클릭 | 버튼 비활성화, "학습이 완료되었습니다" 표시 |
| 자동 재생 중 [초기화] 클릭 | 자동 재생 즉시 중단 후 초기화 |
| MSE가 수렴하여 변화 없음 | 오차 그래프 수평 유지, 카드 수치 변화 없음으로 표시 |
| 자동 재생 중 learning rate 변경 | 자동 재생 즉시 중단, 초기화 자동 실행 후 새 learning rate 적용 |
| 학습 진행 중 learning rate 변경 | 초기화 자동 실행 후 새 learning rate 적용 |

## 9. 표시할 데이터 및 상태값

| 상태값 | 설명 |
|--------|------|
| `step` | 현재 스텝 수 |
| `w` | 현재 기울기 파라미터 |
| `b` | 현재 절편 파라미터 |
| `prevW` | 이전 스텝의 w (카드 비교 표시용) |
| `prevB` | 이전 스텝의 b (카드 비교 표시용) |
| `gradW` | 현재 스텝의 ∂L/∂w |
| `gradB` | 현재 스텝의 ∂L/∂b |
| `mse` | 현재 MSE |
| `prevMse` | 이전 스텝 MSE |
| `mseHistory` | 전체 MSE 기록 (오차 그래프용) |
| `isPlaying` | 자동 재생 여부 |
| `learningRate` | 현재 선택된 learning rate (`0.0001 / 0.001 / 0.01`, 기본값 `0.001`) |

※ step=0 진입 시 prevMse는 표시하지 않는다. 1스텝 이후부터 prevMse는 직전 스텝의 MSE값으로 표시한다. "—"로 표시하지 않는다.
