## 5. 캔버스 좌표계 정의

### 논리 좌표계 (데이터 공간)
- x축 (가로): x1, 좌 → 우 증가
- y축 (세로): x2, 아래 → 위 증가 (수학적 표준)
- x1 범위: `0 ~ 10` (정수)
- x2 범위: `0 ~ 10` (정수)

### 축 레이블 렌더링 규칙
- 캔버스 x축 끝(우측 하단)에 `x1` 레이블 표시
- 캔버스 y축 끝(좌측 상단)에 `x2` 레이블 표시
- 레이블 색상: `#6B7280` (보조 텍스트)

### 픽셀 좌표계 (캔버스 DOM)
- 브라우저 캔버스는 y축이 위 → 아래 증가 (반전)
- **구현 시 반드시 y축 변환 적용**: `pixelY = canvasHeight - (logicalY / yRange × canvasHeight)`

### 결정 경계선 렌더링 규칙
- 결정 경계는 논리 좌표계 기준으로 계산 후 픽셀 좌표로 변환하여 렌더링
- threshold는 0.5로 고정

## 6. 인터랙션 흐름

1. 페이지 진입 시 캔버스 빈 상태, 클래스 A 토글 기본 선택
2. **점 찍기**
   - 클릭 위치를 정수 좌표로 스냅 (반올림) — 소수점 없이 1단위 격자에 점 찍힘
   - 클래스 A 토글 선택 후 캔버스 클릭 → 파란 점 추가
   - 클래스 B 토글 선택 후 캔버스 클릭 → 빨간 점 추가
   - 점 옆에 좌표 항상 표시 — 클래스 A: `(x1, x2)` 파란 텍스트, 클래스 B: `(x1, x2)` 빨간 텍스트, 12px
   - 두 클래스 점이 각각 2개 이상일 때 결정 경계 계산 및 표시
   - 결정 경계 표시와 동시에 클래스 영역 색 표시
   - 결정 경계 수식 표시판 업데이트
3. **점 제거**
   - 마우스가 기존 점 위에 올라오면 커서가 pointer로 변경
   - 점 위에 "클릭하여 제거" 툴팁 표시 (점 바로 위, 작은 회색 텍스트)
   - 점 위에서 클릭 시 해당 점 제거
   - 점 제거 후 결정 경계/수식 표시판 즉시 재계산
   - 점 제거 후 조건 미충족 시 결정 경계/영역색/수식 표시판 초기화
4. **확률 표시**
   - 마우스가 캔버스 위를 이동할 때 해당 위치의 예측 확률 실시간 표시
   - 확률 표시판: "이 위치가 클래스 B일 확률: 0.73"
   - 시그모이드 곡선 패널에서 현재 위치의 선형 출력값이 곡선 위 점으로 표시
5. **초기화 버튼 클릭** → 모든 점 제거, 경계선/영역색 제거, 수식 표시판 초기화

## 7. 시그모이드 곡선 패널 상세

- 소형 그래프 (캔버스 우측 배치)
- x축: 선형 출력값 (로그 오즈)
- y축: 확률 (0 ~ 1)
- 곡선: 시그모이드 함수 `σ(z) = 1 / (1 + e⁻ᶻ)` 고정 표시
- threshold 수평선: y=0.5 위치에 점선으로 고정 표시
- 마우스 hover 시: 현재 위치의 선형 출력값이 곡선 위 점으로 표시
- 패널 상단 설명: "선형 출력값이 S자 곡선을 통해 0~1 사이 확률로 변환됩니다"

---

## 7-1. 결정 경계 수식 표시판 상세

- **1번 줄**: 원래 수식(좌) + `=>` + 숫자 대입된 수식(우)
  - 좌: `w1·x1 + w2·x2 + b = 0` (회색 텍스트 `#6B7280`)
  - 우: `0.82·x1 + (-0.45)·x2 + (-1.23) = 0` (강조색 `#6366F1`)

- **2번 줄**: 그래프 해석용 수식(좌) + `=>` + 완전히 계산된 숫자(우)
  - 좌: `x2 = -(w1/w2)·x1 - (b/w2)` (회색 텍스트 `#6B7280`)
  - 우: `-(w1/w2)`와 `-(b/w2)`를 각각 나눗셈까지 계산해 단일 숫자로 표현
    예: `x2 = 1.82·x1 - 2.73` (강조색 `#6366F1`)

- 점이 부족해 계산 불가 시: 우측 "—" 표시
- 수치: 소수점 2자리 고정

## 8. 엣지 케이스

| 상황 | 처리 방식 |
|------|-----------|
| 점이 0개 | 결정 경계/영역색 미표시, 확률 표시판 "—", 수식 표시판 "—" |
| 한 클래스만 점이 있는 경우 | 결정 경계 미표시, 안내 문구: "두 클래스 모두 점을 찍어보세요" |
| 한 클래스의 점이 1개인 경우 | 결정 경계 미표시, 동일 안내 문구 표시 |
| 점 20개 초과 | 추가 무시, "최대 20개까지 찍을 수 있어요" 안내 |
| 캔버스 경계 밖 클릭 | 점 추가 무시 |
| 두 클래스가 완전히 선형 분리 불가능한 경우 | 최선의 경계선 표시, 별도 안내 없음 |
| 동일한 좌표에 다른 클래스 점이 이미 있는 경우 | 점 추가 무시 |
| 점 제거 후 한 클래스 점이 1개 이하가 되는 경우 | 결정 경계/영역색/수식 표시판 초기화, 안내 문구: "두 클래스 모두 점을 찍어보세요" |

## 9. 표시할 데이터 및 상태값

| 상태값 | 설명 |
|--------|------|
| `pointsA` | 클래스 A 점들의 정수 좌표 배열 |
| `pointsB` | 클래스 B 점들의 정수 좌표 배열 |
| `selectedClass` | 현재 찍을 클래스 (`A` 또는 `B`) |
| `w1` | 결정 경계의 x1 가중치 |
| `w2` | 결정 경계의 x2 가중치 |
| `b` | 결정 경계의 편향값 |
| `threshold` | 분류 임계값 (고정값 `0.5`, 변경 불가) |
| `hoverProb` | 마우스 위치의 예측 확률 (확률 표시판용) |
| `hoverLogit` | 마우스 위치의 선형 출력값 (시그모이드 패널용) |
