경사하강법 시뮬레이션: Batch ↔ Mini-batch/SGD (올바른 epoch 정의)

모드
batch size 1
epoch 진행: 0 / 0 BATCH

데이터 + 회귀 직선 (y = w·x + b)

왼쪽: 현재 w,b로 그려지는 직선이 데이터에 맞춰 이동
Mini/SGD 모드에서는 이번 step에서 선택된 샘플(들)을 빨간 점으로 강조

Convex Loss 곡선: L(w) = MSE(w, b 현재값 고정)

오른쪽: 현재 b에서의 단면(U자 곡선) 위로 점이 이동
Batch는 부드럽고, Mini/SGD는 수렴 근처에서도 지그재그가 관찰됨
