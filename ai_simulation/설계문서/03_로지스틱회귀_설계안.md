## 3. UI 컴포넌트 목록

1. **상단 헤더**: 페이지 제목 + 한 줄 정의 + `← 목록으로` 뒤로가기 링크
2. **캔버스 영역**: 점 찍기 + 결정 경계선 + 클래스 영역 색 표시
3. **클래스 선택 토글**: [클래스 A] / [클래스 B] — 현재 찍을 클래스 선택
4. **확률 표시판**: 마우스가 캔버스 위에 있을 때 해당 위치의 예측 확률 실시간 표시
5. **시그모이드 곡선 패널**: 선형 출력값 → 확률 변환 과정을 소형 그래프로 표시
6. **Threshold 슬라이더**: 0.0 ~ 1.0, 기본값 0.5
7. **Threshold 표시판**: 현재 threshold 값 + 경계 변화 실시간 반영
8. **초기화 버튼**: 캔버스의 모든 점 제거
9. **하단 설명 문구**

## 5. 캔버스 좌표계 정의

### 논리 좌표계 (데이터 공간)
- x축: 좌 → 우 증가
- y축: 아래 → 위 증가 (수학적 표준)
- x 범위: `0.0 ~ 10.0`
- y 범위: `0.0 ~ 10.0`

### 픽셀 좌표계 (캔버스 DOM)
- 브라우저 캔버스는 y축이 위 → 아래 증가 (반전)
- **구현 시 반드시 y축 변환 적용**: `pixelY = canvasHeight - (logicalY / yRange × canvasHeight)`

### 결정 경계선 렌더링 규칙
- 결정 경계는 논리 좌표계 기준으로 계산 후 픽셀 좌표로 변환하여 렌더링
- threshold 변경 시 경계선 위치 즉시 업데이트

## 6. 인터랙션 흐름

1. 페이지 진입 시 캔버스 빈 상태, 클래스 A 토글 기본 선택
2. **점 찍기**
   - 클래스 A 토글 선택 후 캔버스 클릭 → 파란 점 추가
   - 클래스 B 토글 선택 후 캔버스 클릭 → 빨간 점 추가
   - 두 클래스 점이 각각 2개 이상일 때 결정 경계 계산 및 표시
   - 결정 경계 표시와 동시에 클래스 영역 색 표시
3. **확률 표시**
   - 마우스가 캔버스 위를 이동할 때 해당 위치의 예측 확률 실시간 표시
   - 확률 표시판: "이 위치가 클래스 B일 확률: 0.73"
   - 시그모이드 곡선 패널에서 현재 위치의 선형 출력값이 곡선 위 점으로 표시
4. **Threshold 슬라이더 조작**
   - 슬라이더 이동 시 결정 경계선 위치 실시간 업데이트
   - 클래스 영역 색 범위 실시간 업데이트
   - threshold 값 표시판 실시간 업데이트
5. **초기화 버튼 클릭** → 모든 점 제거, threshold 0.5로 초기화, 경계선/영역색 제거

---

## 7. 시그모이드 곡선 패널 상세

- 소형 그래프 (캔버스 우측 배치)
- x축: 선형 출력값 (로그 오즈)
- y축: 확률 (0 ~ 1)
- 곡선: 시그모이드 함수 `σ(z) = 1 / (1 + e⁻ᶻ)` 고정 표시
- threshold 수평선: 현재 threshold 값에 해당하는 y 위치에 점선으로 표시
- 마우스 hover 시: 현재 위치의 선형 출력값이 곡선 위 점으로 표시
- 패널 상단 설명: "선형 출력값이 S자 곡선을 통해 0~1 사이 확률로 변환됩니다"

## 8. 엣지 케이스

| 상황 | 처리 방식 |
|------|-----------|
| 점이 0개 | 결정 경계/영역색 미표시, 확률 표시판 "—" |
| 한 클래스만 점이 있는 경우 | 결정 경계 미표시, 안내 문구: "두 클래스 모두 점을 찍어보세요" |
| 한 클래스의 점이 1개인 경우 | 결정 경계 미표시, 동일 안내 문구 표시 |
| 점 20개 초과 | 추가 무시, "최대 20개까지 찍을 수 있어요" 안내 |
| 캔버스 경계 밖 클릭 | 점 추가 무시 |
| 두 클래스가 완전히 선형 분리 불가능한 경우 | 최선의 경계선 표시, 별도 안내 없음 |
| threshold = 0.0 | 전체 영역이 클래스 B로 분류, 경계선 캔버스 최하단 표시 |
| threshold = 1.0 | 전체 영역이 클래스 A로 분류, 경계선 캔버스 최상단 표시 |

## 9. 표시할 데이터 및 상태값

| 상태값 | 설명 |
|--------|------|
| `pointsA` | 클래스 A 점들의 좌표 배열 |
| `pointsB` | 클래스 B 점들의 좌표 배열 |
| `selectedClass` | 현재 찍을 클래스 (`A` 또는 `B`) |
| `weights` | 로지스틱 회귀 가중치 (결정 경계 계산용) |
| `threshold` | 현재 분류 임계값 (기본값 `0.5`) |
| `hoverProb` | 마우스 위치의 예측 확률 (확률 표시판용) |
| `hoverLogit` | 마우스 위치의 선형 출력값 (시그모이드 패널용) |
