동적 타입 — 타입 선언 없음
차이점

Python은 변수에 타입을 선언하지 않습니다. ML에서는 같은 변수에 int → float → 배열이 담기는 경우가 많습니다.

☕ Java
int count = 0;
double lr = 0.001;
String name = "model";
🐍 Python
count = 0
lr = 0.001
name = "model"

들여쓰기 블록 — 중괄호 없음
차이점

{} 대신 스페이스 4칸으로 블록을 구분합니다. 학습 루프에서 들여쓰기 실수가 가장 흔한 버그입니다.

☕ Java
for (int i=0; i<5; i++) {
    if (i % 2 == 0) {
        System.out.println(i);
    }
}
🐍 Python
for i in range(5):
    if i % 2 == 0:
        print(i)
# 블록 끝 표시 없음
⚠️탭과 스페이스를 섞으면 IndentationError. 스페이스 4칸만 사용하세요.

배열 생성 & shape
필수

.shape는 배열 크기를 튜플로 반환합니다. ML 디버깅의 90%는 shape 확인입니다.

☕ Java
float[][] mat = new float[3][4];
// mat.length    → 3
// mat[0].length → 4
🐍 NumPy
mat = np.zeros((3, 4))
mat.shape  # (3, 4)
mat.dtype  # float64
mat.ndim   # 2
